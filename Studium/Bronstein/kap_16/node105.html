2. Methode der mittleren Ziffern von Quadraten:
Eine einfache Methode zur Erzeugung von Zufallszahlen wurde von J. V. NEUMANN vorgeschlagen. Sie wird auch Methode der mittleren Ziffern von Quadraten genannt und geht von einer Dezimalzahl aus, die aus 2n Ziffern besteht. Dann bildet man z2 und erhält eine Dezimalzahl, die aus 4n Ziffern besteht. Von diesen streicht man die ersten und die letzten n Ziffern weg, so daß man wieder eine 2n-ziffrige Dezimalzahl erhält. Diese Vorgehensweise wird wiederholt. So erhält man 2n-stellige Dezimalzahlen, die als Zufallszahlen benutzt werden können. Er lieferte mehr kleine Werte, als in der Regel gebraucht wurden. Deshalb wurden verschiedene andere Methoden entwickelt.
